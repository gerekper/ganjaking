YITH WooCommerce Category Accordion provides a convenient and detailed way to navigate through all you shop's categories.

== Description ==

All categories at your fingertips.

The freedom to explore and track all the categories you need with at a glance: YITH WooCommerce Category Accordion offers a clear and practical toggle accordion to choose exactly where you want to go, both from WordPress and WooCommerce categories.

**Main features:**

* Show WooCommerce categories
* Show WordPress categories
* Highlight current category
* Show WooCommerce subcategories
* Show WordPress subcategories

For a more detailed list of options and features of the plugin, please look at the [official documentation](http://yithemes.com/docs-plugins/yith-woocommerce-category-accordion/ "Yith WooCommerce Category Accordion official documentation").

= Installation =

Once you have installed the plugin, you just need to activate it to enable it.

= Configuration =

YITH WooCommerce Category Accordion will add a new submenu called "Category Accordion" in "YIT Plugins" menu. From there, you can configure all plugin settings.
You will be able to add your accordion(s) in your sidebar thanks to a useful widget.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH WooCommerce Category Accordion` from Plugins page
