=== WooCommerce Chase Paymentech Gateway ===
Author: skyverge
Tags: woocommerce
Requires PHP: 7.0
Requires at least: 5.2
Tested up to: 6.0.1

Accept credit cards in WooCommerce with the Chase Paymentech gateway

See https://docs.woocommerce.com/document/woocommerce-chase-paymentech// for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-chase-paymentech' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
